  • Overview

    O GitHub é uma das plataformas mais importantes para o desenvolvimento colaborativo de código, permitindo versionamento, colaboração e armazenamento de projetos. Ele é amplamente utilizado por desenvolvedores para contribuir com projetos open-source, compartilhar códigos e trabalhar em equipe. Neste curso, você aprenderá desde o básico até funcionalidades mais avançadas, como o uso de branches, pull requests e GitHub Actions.
