  • Overview

    JavaScript (JS) é uma das linguagens de programação mais populares do mundo, amplamente utilizada para desenvolvimento web, tanto no lado do cliente quanto no servidor. Neste curso, você aprenderá desde os conceitos básicos de JS até tópicos avançados, como manipulação de DOM, async/await e APIs modernas. Desenvolva suas habilidades em criar aplicações dinâmicas e interativas com JavaScript.
